💡 시멘틱 태그란?

: 시멘틱 태그는 HTML 요소를 의미에 맞게 사용한다는 뜻입니다. 예를들어 header는 머리글 영역을 의미합니다.

🔎 시멘틱 태그를 쓰는 이유

: 시멘틱 태그를 사용하면, 검색엔진최적화(SEO)와 접근성 향상에 도움이 됩니다. 그리고 유지보수를 하기에 더 용이합니다.
